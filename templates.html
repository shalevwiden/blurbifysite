What are Blurb templates?

Blurb templates come with preassigned, non changable usernames, names, and profile pictures, but with updateable content fields to create a variety of Blurbs. Each Blurb template comes with default color styling which can be changed.

In addition, each Blurb template comes with a unique default flyingimage. The flying image can also be changed.



More Information on Templates:

  • All templates have a pinkish redish profile picture border. Seeing that Border indicates that it is a Blurb template.
  • All templates have their own hashtags so you can see where that template was used across any social platforms
  • You can request a template be added through this link: template request form
